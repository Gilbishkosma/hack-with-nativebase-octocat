We have kept the React Native Project in **mobile/** and the Next Js Project in the **web/** folder

## Instructions to run React Native project

1. cd mobile/
2. npm install
3. npm run android   // For Android

   npm run ios       // For IOS


## Instructions to run Next Js project

1. cd web/
2. npm install 
3. npm run dev
